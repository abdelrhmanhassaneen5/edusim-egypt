# EduSim Egypt 🚀

**محاكي المجتمعات التعليمية المصرية بالذكاء الاصطناعي**

مشروع ذكاء اصطناعي متقدم يحاكي **آلاف الشخصيات الافتراضية** (طلاب + أولياء أمور + معلمين) في المراحل الابتدائية والإعدادية والثانوية.

### الهدف
بدل ما تجرب فكرة جديدة أو قرار أو اختبار أو فيديو شرح في الواقع وتخاطر بنتيجته، المشروع ينفذ الفكرة **داخل مجتمع افتراضي** ويعطيك تقرير كامل:
- المزايا والعيوب
- مين هيتقبله ومين هيرفضه
- المشاكل المتوقعة
- توصية نهائية: **نشر – تعديل – إلغاء**

### المميزات الرئيسية
- كل شخصية لها ذاكرة + مشاعر + قرارات بشرية
- محاكاة حية للواتساب والمدارس والبيوت
- اللهجة المصرية الطبيعية
- تقارير ذكية بالعربي
